##  Project Overvie
Lingwu is an AI-powered intelligent agent platform that transforms unstructured group chat transcripts and multi-modal documents into structured, reusable knowledge bases. It was developed to redefine human-agent collaboration through a dynamic, knowledge-driven LLM Agent framework. This codebase is solely for poster demonstration purposes. Ownership of the project belongs to the responsible person. I am only responsible for a small part of it.

##  My Specific Contributions
While the platform was built collectively, my core responsibilities and technical focus included:
1.  **Workflow Orchestration**: Designed and debugged the multi-stage Agent interaction logic.
2.  **Knowledge Base Construction**: Managed the structuring and indexing of multi-modal data to ensure high-precision retrieval.
3.  **Prompt Engineering**: Iteratively optimized prompts to enhance the Agent's reasoning, memory retention, and planning capabilities.
